# Ciphers

Simple tool for encoding and decoding text using various cipher algorithms.

## Features

- **Direct Converter**: Instant translation between any two supported formats (e.g., Plaintext to Caesar).
- **Chain Converter**: Advanced mode for decoding from one cipher and immediately encoding into another.
- **Cipher Gallery**: A collection of standalone tools for specific algorithms.
- **State Persistence**: Saves your last used configuration and input text locally in your browser.
- **Cross-Cipher Compatibility**: Swap inputs, outputs, and algorithms with a single click.
